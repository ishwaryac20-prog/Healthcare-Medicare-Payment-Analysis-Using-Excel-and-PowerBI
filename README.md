# 📖 Project Overview

Healthcare organizations generate millions of Medicare payment records, making it difficult to identify payment trends and provider performance.

This project leverages **Microsoft Excel** and **Power BI** to transform raw Medicare payment data into a fully interactive business intelligence dashboard that helps stakeholders monitor payments, evaluate provider performance, and identify regional trends.

✨ **This dashboard enables users to:**

✔️ Analyze Medicare Payments across States & Cities

✔️ Compare Submitted Charges vs Medicare Payments

✔️ Evaluate Provider Performance

✔️ Analyze Specialty-wise Payment Distribution

✔️ Explore Interactive Filters & Visualizations

# 🔍 Key Insights

📈 **Total Medicare Payment:** **$41.18 Million**

💰 **Total Submitted Charges:** **$179.70 Million**

🏙️ Metropolitan areas generated the highest Medicare payments.

👁️ Ophthalmology contributed the highest Medicare reimbursement among all specialties.

📍 Medicare payments vary significantly across different states and providers.

📊 Higher service volumes resulted in increased Medicare reimbursements.

💸 Submitted Charges were considerably higher than Medicare Payments due to reimbursement policies.
